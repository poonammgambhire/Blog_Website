# 📝 Blog Website

A full-stack Blog application built with MERN Stack.

## 🚀 Features
- User Authentication (JWT + bcrypt)
- Create, Read, Update, Delete Posts
- Image Upload (Multer + Cloudinary)
- Comments System
- Search & Filter Posts
- Responsive Design

## 🛠️ Tech Stack
**Frontend:** React.js, Redux, Tailwind CSS

**Backend:** Node.js, Express.js, MongoDB

**Other:** JWT, bcrypt, Cloudinary, Multer

## ⚙️ Installation

```bash
# Clone the repo
git clone https://github.com/poonammgambhire/Blog_Website.git

# Install dependencies
cd Blog_Website
npm install

# Run
npm start
```
